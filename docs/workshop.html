About ProControls

ProControls is a comprehensive UI control library built specifically for p5.js that brings professional, hardware-inspired interface elements to your interactive sketches. Whether you're creating generative art, music production tools, data visualizations, games, or educational applications, ProControls provides everything you need to build polished, responsive user interfaces with minimal code.

Inspired by professional audio hardware interfaces, ProControls combines modern design principles with the simplicity of p5.js. The library includes 24+ ready-to-use controls, 8 beautiful built-in themes, and a thoughtful API that lets you focus on your creative vision instead of reinventing UI components.

The Problem It Solves

Building interactive interfaces in p5.js has traditionally required writing hundreds of lines of code. You'd need to:

  • Manually draw rectangles, circles, and text for each control
  • Implement complex mouse event handling and hit detection
  • Manage control state and callback functions yourself
  • Design consistent styling across multiple controls
  • Handle edge cases like text overflow, scrolling, and tooltips

ProControls eliminates all of this. Instead of building from scratch, you declare what controls you want, and the library handles rendering, interaction, state management, and visual polish automatically.

Code Comparison

Building a Simple Interface: With vs Without ProControls

❌ Without ProControls

~150 lines of manual code

// Manual state management let sliderX = 100, sliderY = 100; let sliderW = 80, sliderH = 15; let sliderValue = 0.5; let isSliderDragging = false; // Manual rendering function draw() { // Draw background fill(240); rect(sliderX, sliderY, sliderW, sliderH); // Draw handle let handleX = sliderX + sliderValue * sliderW; fill(100); circle(handleX, sliderY + sliderH/2, 10); // Draw label fill(0); text('Value: ' + sliderValue.toFixed(2), sliderX, sliderY - 10); } // Manual mouse handling function mouseDragged() { if (mouseX > sliderX - 5 && mouseX < sliderX + sliderW + 5 && mouseY > sliderY - 5 && mouseY < sliderY + sliderH + 5) { isSliderDragging = true; } if (isSliderDragging) { sliderValue = constrain( (mouseX - sliderX) / sliderW, 0, 1 ); } } function mouseReleased() { isSliderDragging = false; }

✅ With ProControls

~3 lines of code

const slider = new AnalogSlider({ x: 100, y: 100, onChange: (v) => { console.log(v); } });

Quick Start Guide

Installation

Add one script tag to your HTML:

<script src="https://path/to/ProControls.js"></script>

Your First Control

Create a slider in your p5.js sketch:

function setup() { createCanvas(400, 400); new AnalogSlider({ x: 20, y: 20, label: 'Brightness', onChange: (val) => { background(val * 255); } }); } function draw() { proControlBackground(); }
