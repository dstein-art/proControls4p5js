Key Topics Deep Dive

1. Understanding the ProControls API

ProControls follows consistent patterns that make it intuitive to learn:

  • Creating controls with options objects
  • Callbacks: onChange, onRelease, onClick, onSelect
  • Getting and setting control values programmatically
  • Removing and updating controls
  • The proControls() array for managing instances

2. Working with Callbacks

Master the event system that connects controls to your sketch:

const slider = new AnalogSlider({ x: 20, y: 15, // Called while user drags onChange: (value) => { background(100 + value * 155); }, // Called when user releases onRelease: (value) => { console.log('Final:', value); } });

3. Unique Features

What makes ProControls special:

  • Spring Animations — Smooth returns to default values
  • Multi-value Controls — Single controls managing multiple parameters
  • Real-time Visualization — Graphs and meters that update live
  • Formatted Markup — Rich text with links and formatting
  • Smart Scrolling — Containers manage overflow automatically
  • Panel System — Create resizable, movable interfaces

The Power of ProControls

Without ProControls: ~150 lines of code

// Manual state management let sliderX = 100, sliderY = 100; let sliderW = 80, sliderH = 15; let sliderValue = 0.5; let isSliderDragging = false; // Manual rendering function draw() { fill(240); rect(sliderX, sliderY, sliderW, sliderH); let handleX = sliderX + sliderValue * sliderW; fill(100); circle(handleX, sliderY + sliderH/2, 10); } // Manual mouse handling + more...

With ProControls: ~3 lines of code

const slider = new AnalogSlider({ x: 100, y: 100, onChange: (v) => console.log(v) });

Result: Write 50x less code. Get professional rendering, mouse handling, value constraints, visual feedback, callbacks, and themes automatically.

Getting Started Now

1. Include the Library

<script src="path/to/ProControls.js"></script>

2. Create Your First Control

function setup() { createCanvas(400, 400); new AnalogSlider({ x: 20, y: 20, label: 'Brightness', onChange: (v) => { background(v * 255); } }); } function draw() { proControlBackground(); }
