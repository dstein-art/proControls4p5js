Menu PopupMenu

Menu Input

A menu bar that runs either along the top (orientation: 'horizontal') or down the left side (orientation: 'vertical'). Items are defined as an array of strings; replace any string with an array to create a submenu — the first element of the array becomes the top-level label and the remaining elements are the submenu entries. Clicking a leaf item fires onChange; clicking a submenu parent toggles it open or closed.

new Menu(opts)

Options

OptionTypeDefaultDescription
itemsArray[]Menu definition. Each entry is a string (leaf item) or a string[] (submenu) where array[0] is the top-level label and array[1…n] are the submenu entries.
orientationstring'top'Where the bar sits and which way submenus open: 'top' — bar across the top, submenus open downward. 'bottom' — bar across the bottom, submenus open upward. 'left' — bar down the left side, submenus open rightward. 'right' — bar down the right side, submenus open leftward. 'popup' — floating vertical menu positioned at x, y; submenus and their arrow indicators automatically flip left or right depending on which half of the canvas the menu sits in.
x, ynumber0, 0Canvas position of the top-left corner of the menu bar.
widthnumberautoBar width. Horizontal: defaults to the total of all item widths. Vertical: defaults to the widest item label.
onChangefunctionnullonChange({label, path}[, control]) — called when a leaf item is selected. label is the selected item's text; path is an array from root to selection, e.g. ['Edit', 'Undo'].
disabledbooleanfalseSuppresses all interaction and draws a disabled overlay.
themeobject{}Partial theme override. See Themes.

Properties

PropertyDescription
.itemsRead/write. Reassign to change the menu structure. Set _layoutDirty = true after changing to force a layout rebuild.

PopupMenu Input

A convenience wrapper around Menu pre-configured with orientation: 'popup'. Positions a floating vertical menu at the given x, y coordinate. Submenus and their arrow indicators automatically flip left or right depending on which half of the canvas the menu sits in. Accepts all the same options as Menu.

new PopupMenu(opts)
Inherits all Menu options. orientation is fixed to 'popup'.
